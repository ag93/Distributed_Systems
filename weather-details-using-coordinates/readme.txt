1. Overview
Create a client application that will connect to the National Weather Service website using HTTP and XML and/or SOAP and display current weather conditions.

2. Development of Project
(i)   Java used as the programming language.
(ii)  Netbeans IDE used for programming 

3. Implementation 
i)    Client connects to weather server
ii)   Client uses SOAP and/or XML to retrieve the information
iii)  Client displays correct weather information
iv)   Session disconnects after fetching data
v)    Client reconnects on command for a refresh
vi)   Comments in code

4. Software Requirements
(i)   Windows OS
(ii)  Netbeans IDE.
(iii) Java jdk v7 and above.

5. Steps to Run the code
In the Zip file, I have included the entire Netbeans project.
(i)   Open the project in Netbeans.
(ii)  Run the gui.java file.
(iii) Enter Latitude and Longitude and Hit Submit. Hit refresh to reolad the information.
